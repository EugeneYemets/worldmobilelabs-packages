# worldmobile-rs (Axum + Utoipa)
Rust-бекенд зі Swagger UI: `/countries` (список країн) і `/health`.
- Swagger: http://localhost:8000/docs
- Налаштування: `.env` (див. приклад у корені)

## Запуск
cargo run
